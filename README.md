# learn-router
这是一个简单的教学项目，用来示范新版本Angular路由的基本用法。

每一个功能点的演示代码都在对应的分支上：

  - basic分支：路由的基本配置
  - async-module分支：懒加载模块
  - nested-router分支：N层嵌套路由
  - shared-module分支：共享模块
  - router-events分支：路由事件
  - router-params分支：在路由上传递和获取参数的各种方式
  - guard分支：演示路由守卫的用法
  - preload-module分支：演示模块的预加载特性
